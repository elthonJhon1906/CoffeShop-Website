# CoffeeShop Website

Situs statis bertema kedai kopi (Filosofi Kopi) yang dibangun dengan HTML, CSS, dan JavaScript. Mencakup landing page dengan navigasi anchor, katalog menu, produk biji kopi, testimoni pelanggan, formulir kontak yang dilengkapi embed peta, serta bagian blog dan footer sosial.

Nama repository di GitHub adalah `CoffeShop-Website`, sehingga slug dan URL live memakai e tunggal sesuai nama repo.

- Live demo: https://elthonjhon1906.github.io/CoffeShop-Website/
  - Untuk fork, sesuaikan: https://[YOUR_USERNAME].github.io/CoffeShop-Website/

## Fitur Utama
- Header dengan logo, navigasi anchor, serta tombol pencarian, keranjang, dan menu samping yang ditoggle lewat `js/main.js`.
- Section hero, About (profil pendiri + lokasi), Menu (kartu harga minuman), Products (grid produk biji kopi), Reviews, Contact (form dan Google Maps embed), Blogs, dan Footer sosial.
- Styling kustom di `css/style.css`; aset visual ada di folder `images/`.

## Struktur Proyek
- `index.html` – markup utama dan referensi ke ikon Font Awesome, CSS, dan JavaScript.
- `css/style.css` – gaya untuk layout, tipografi, warna, dan responsivitas.
- `js/main.js` – logika toggle untuk pencarian, keranjang, dan navbar saat scroll.
- `images/` – logo, foto produk, pendiri, dan ilustrasi blog.

## Menjalankan Secara Lokal
1. Clone atau unduh repo ini.
2. Buka `index.html` langsung di browser, atau layani lewat server statis (mis. `npx serve`).
